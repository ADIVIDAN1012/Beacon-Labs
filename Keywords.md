# Dyno Programming Language - Keywords

---

## Universal User-Oriented Programming (UOP)
| Dyno Keyword  | Meaning                                      |
|---------------|----------------------------------------------|
| use           | Import modules, share resources              |
| fetch         | Read data                                    |
| modify        | Write data                                   |
| ask           | Get user input                               |
| output        | Display output to the user                   |

---

## Access Control
| Dyno Keyword  | Meaning                                      |
|---------------|----------------------------------------------|
| avail         | Public                                       |
| shel          | Protected                                    |
| covnito       | Private                                      |
| internal      | Package-level access                         |
| expose        | Expose visibility                           |

---

## Classes & Objects
| Dyno Keyword  | Meaning                                      |
|---------------|----------------------------------------------|
| blueprint     | Class                                        |
| adopt         | Inherit / Extend                             |
| own           | Self / This                                  |
| father        | Base class                                   |
| child         | Derived class                                |

---

## Functions
| Dyno Keyword  | Meaning                                      |
|---------------|----------------------------------------------|
| spec          | Function declaration                         |
| send          | Return value                                 |
| funcall       | Call a function                              |
| den           | Anonymous function                           |

---

## Flow Control
| Dyno Keyword  | Meaning                                      |
|---------------|----------------------------------------------|
| check         | if                                           |
| alter         | elif                                         |
| altern        | else                                         |
| attempt       | try                                          |
| trap          | catch                                        |
| conclude      | finally                                      |
| stop          | break / exit                                 |

---

## Loops
| Dyno Keyword                | Meaning                                      |
|-----------------------------|----------------------------------------------|
| until                       | while                                        |
| traverse V from X to Y      | for loop                                     |
| scope(A, B)                 | Range function                               |

---

## Logic Operators
| Dyno Keyword  | Meaning                                      |
|---------------|----------------------------------------------|
| both          | and                                          |
| either        | or                                           |
| deny          | not                                          |

---

## Data Types & Structures
| Dyno Keyword  | Meaning                                      |
|---------------|----------------------------------------------|
| list          | List / Array                                 |
| group         | Set                                          |
| dict          | Dictionary                                   |
| bind          | Pair                                         |
| seal          | Tuple                                        |

---

## Type Handling

|------------------|----------------------------------------------|

## Modules & Importing
| Dyno Keyword  | Meaning                                      |
|---------------|----------------------------------------------|
| use           | Import                                       |
| share         | Export                                       |
| toolkit       | Module                                       |
| embed         | From                                         |
| by            | As alias                                     |

## File Operations

|---------------|----------------------------------------------|

| unlock        | Open file                                    |
| seal          | Close file                                   |

## Concurrency

## Events
