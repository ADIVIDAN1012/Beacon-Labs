The Problem with Coding

Hieroglyphics?

Feature: Coding in English

Readable Syntax

Beacon removes the cryptic symbols. It uses words like traverse, from, and show.

This makes the code self-explanatory. You can read it out loud and it makes sense.

# A simple loop in Beacon traverse i from 1 to 10: show "Hello Family!" done

Clean & Readable

New in v2.0: "Packs"

# Grouping items together firm groceries = pack( "Apples", "Milk", "Bread" )

Organized Data

With version 2.0, I introduced Packs.

This is a new way to group related items together, making it easier to manage lists of data like shopping lists, user names, or inventory.

Smart Type Detection

Intelligent Parsing

Beacon knows that 42 is a number and "Hello" is text automatically.

The user doesn't have to worry about defining types. The language handles the details.
